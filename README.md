# 全端统一开发模板脚手架

## 使用

````sh
npx @naice/create

or

npx @naice/create [type] # h5 | react | express | lib

````


## 说明

1. web： vite+ts+vue3 移动端开发模板
2. react： vite+ts+react 开发模板
3. express： node+express+ts 开发模板
4. lib：工具库开发模板
5. vue-component： vue组件库开发模板（coming）
6. react-component：react组件库开发模板（coming）
